Встановити додаток
Радіо Світло

radio-svitlo.com

Додай Радіо Світло на екран «Додому» для швидкого доступу в один клік.

1 Натисніть кнопку «Поділитися» (квадрат зі стрілкою) внизу Safari.
2 Оберіть «Додати на початковий екран» і натисніть «Додати».
Android (Chrome)

Встановити додаток
Радіо Світло

radio-svitlo.com

Додай Радіо Світло на головний екран Android для швидкого доступу в один клік.

Або вручну через меню Chrome

1 Натисніть меню у верхньому правому куті Chrome.
2 Оберіть «Додати на головний екран» або «Встановити додаток».
3 Натисніть «Встановити» у діалозі.

Радіо Світло

Версія для Windows

Нативний додаток для Windows. Завантаж та встанови — слухай Радіо Світло без браузера.

1 Натисніть кнопку «Завантажити» нижче.
2 Відкрийте завантажений файл RadioSvitlo.exe та запустіть встановлення.
3 Готово — іконка додатку з'явиться на робочому столі.
